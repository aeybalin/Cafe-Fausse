For MAC users:
Clone the files for the project (use the green code button on the main page).
Store them locally on your computer so you can access them easily via Terminal.
Open a window in Terminal and navigate to this project (cd cafefausse)
Navigate to Flask-backend file to set up virtual envirosment (runs well on python 3.11)
Enter python3.11 -m venv env
Enter . env/bin/activate
Install the requirements and run Flask app.
Enter pip install -r requirements.txt
Enter flask run.
To run the front end, open another window in Terminal.
Navigate to this project (cd cafefausse).
Navigate to the reactjs-client folder (cd reactjs-client)
Enter npm install
Enter npm run start
The website should open in a new browser window.
